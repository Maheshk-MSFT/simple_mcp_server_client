# simple_mcp_server_client
Build a simple Server using python

```
PREQ:
===========
$ python3 -m venv env_name
$ source env_name/bin/activate 
$ touch server.py
$ pip3 install mcp[cli]
$ mcp dev server.py
---
$ deactivate
$ pip install --upgrade pip
$ pip uninstall <package_name>
```
